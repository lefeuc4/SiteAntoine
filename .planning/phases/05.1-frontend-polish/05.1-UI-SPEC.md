## Component Contracts (existing component corrections)

### Timeline Year Label — `mon-histoire/page.tsx`

The `titre` field in seeded PageContent documents embeds the year as a prefix: `"2010 — Les Premiers Pas"`.

**Parsing strategy:** Split on ` — ` (space + em-dash + space). If the first segment is a 4-digit year, render it separately.

```
function parseTitre(titre: string): { annee: string | null; nom: string }
Pattern: /^(\d{4})\s*[—–-]\s*(.+)$/
Fallback: { annee: null, nom: titre }
```

**Year label rendering:** `<span className="text-sm font-body text-gris-ardoise block mb-1">{annee}</span>`

**Heading rendering (after year):** `<h3 className="text-xl font-heading font-bold text-bleu-nuit mb-2">{nom}</h3>`

No Payload schema change. No migration needed.

### Focus Rings — ServiceCard, ProgrammeCard, ResultatCard

Cards are `<div>` elements. Add to each card wrapper `<div>`:

```
focus-visible:ring-2 focus-visible:ring-bleu-electrique focus-visible:outline-none
```

No `tabIndex` needed — cards are not interactive in Phase 05.1. The focus-visible classes apply only when the element receives keyboard focus (browser default). This satisfies WCAG 2.1 SC 2.4.7.

### ResultatCard Quote Decorators

Lines 51 and 53: change `text-2xl` to `text-xl`. The decorative `"` and `"` characters render at 28px instead of 32px — still visually prominent, within the declared type scale.
