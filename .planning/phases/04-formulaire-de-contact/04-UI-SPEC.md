## Component Inventory

### 1. Page /contact layout

Structure (top to bottom):
```
<main> max-w-[1280px] mx-auto px-8 lg:px-12 py-12 lg:py-16
  <PageHeading>         h1 — "Contactez-moi"
  <ContactForm>         max-w-[560px] mx-auto
  <WhatsAppBlock>       max-w-[560px] mx-auto mt-8
  <SocialBlock>         max-w-[560px] mx-auto mt-6
</main>
```

### 2. ContactForm

**Fields (in order):**
1. Nom — `<input type="text">`, label "Votre nom", placeholder "Marie Dupont"
2. Email — `<input type="email">`, label "Votre adresse email", placeholder "marie@exemple.fr"
3. Message — `<textarea rows={5}>`, label "Votre message", placeholder "Bonjour Antoine, je souhaite en savoir plus sur..."
4. Honeypot — `<input type="text" name="website" tabIndex={-1} aria-hidden="true">`, hidden via CSS (`position: absolute; opacity: 0; pointer-events: none; width: 0; height: 0;`). Never display:none (some bots skip hidden fields).

**Submit button:** full-width, bg-bleu-electrique, rounded-full, px-8 py-4, font-heading text-sm, "Envoyer mon message". Loading state: spinner icon (lucide `Loader2` animated) + "Envoi en cours..." label, button disabled.

**Field styling:**
- Label: text-sm font-bold text-bleu-nuit, mb-1
- Input/textarea: w-full border border-gris-ardoise/40 rounded-lg px-4 py-3 text-base text-bleu-nuit bg-blanc-pur focus:outline-none focus:ring-2 focus:ring-bleu-electrique focus:border-bleu-electrique transition-all (--transition-fast)
- Error state: border-rouge-erreur focus:ring-rouge-erreur, error message text-sm text-rouge-erreur mt-1

**States:**
- Default: fields empty, button enabled
- Loading: spinner on button, all fields and button disabled
- Success: form replaced by success message (D-08 from CONTEXT.md)
- Error (network/Resend): inline alert above button — "Une erreur est survenue. Veuillez reessayer ou nous contacter directement via WhatsApp."

**Success state (replaces form, D-08):**
- Centered in the same max-w-[560px] container
- lucide `CheckCircle` icon, 48px, color vert-energie (#10B981)
- Heading text-xl font-heading font-bold text-bleu-nuit: "Message envoye !"
- Body text-base text-gris-ardoise: "Merci, je vous reponds dans les 24 heures."
- No redirect, no new page (D-09)

### 3. WhatsApp CTA block (on /contact page)

Container: bg-vert-energie/10 (10% opacity green tint) rounded-xl p-6, flex items-center gap-4

Content:
- lucide `MessageCircle` icon (24px, color #10B981) — WhatsApp brand icon approximation; no external SVG imports
- Text column: heading text-sm font-bold text-bleu-nuit "Preferer WhatsApp ?", body text-sm text-gris-ardoise "Repondez directement en message prive."
- CTA link: `<a href="https://wa.me/{number}?text={encoded_message}">` — renders as pill button, bg-vert-energie text-blanc-pur rounded-full px-5 py-2 text-sm font-heading hover:brightness-110 transition (--transition-fast). Label: "Ouvrir WhatsApp"

Number and pre-filled message sourced from Payload Global ContactSettings at render time.

### 4. Social icons block (on /contact page)

Container: flex items-center gap-4 py-2

Label: text-sm text-gris-ardoise "Retrouvez-moi aussi sur :"

Icons: SVG inline (no external dependency), 24x24px each. Two icons: Instagram, Facebook.
- Default state: color gris-ardoise (#64748B)
- Hover state: color bleu-electrique (#3B82F6), transition-colors --transition-fast
- Each wraps an `<a href={url} target="_blank" rel="noopener noreferrer" aria-label="{Platform name}">` sourced from Payload Global ContactSettings

### 5. Floating WhatsApp button (global, all pages)

**Placement:** Fixed, bottom-6 right-6 (24px from edges). z-50.

**Shape:** Circle, 56px x 56px (w-14 h-14). bg-vert-energie. shadow-lg.

**Icon:** lucide `MessageCircle` (24px, text-blanc-pur).

**Hover:** scale-110, shadow-xl. transition: transform --transition-fast, box-shadow --transition-fast.

**Entrance animation:** Slides up from bottom on mount. CSS: initial `translate-y-4 opacity-0`, animated to `translate-y-0 opacity-100` over --transition-slow (300ms). Implemented via a `useEffect` + state boolean toggling CSS classes.

**Tooltip:** `title="Contactez-moi sur WhatsApp"` attribute. No custom tooltip component needed.

**ARIA:** `aria-label="Ouvrir WhatsApp"`.

**Link:** Same wa.me URL as the /contact block (sourced from ContactSettings Global).

**Implementation location:** Added to `src/app/(frontend)/layout.tsx` as a Client Component import, so it appears on all public pages. The layout fetches ContactSettings once and passes `whatsappNumber` + `whatsappMessage` as props.

## Validation Rules (Zod schema)

Declared here so the planner generates matching client + server validation:

```
nom: z.string().min(2, "Veuillez indiquer votre nom.").max(100)
email: z.string().email("Veuillez saisir une adresse email valide.")
message: z.string().min(10, "Veuillez ecrire votre message.").max(2000)
website: z.string().max(0) // honeypot — must be empty
```
