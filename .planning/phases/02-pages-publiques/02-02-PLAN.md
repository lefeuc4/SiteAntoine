**Lexical JSON format for richText fields:**
All richText fields must use the Lexical editor JSON format:
```typescript
{
  root: {
    type: 'root',
    children: [
      {
        type: 'paragraph',
        children: [
          { type: 'text', text: 'Your text here', format: 0, detail: 0, mode: 'normal', style: '', version: 1 }
        ],
        direction: 'ltr',
        format: '',
        indent: 0,
        textFormat: 0,
        version: 1,
      },
    ],
    direction: 'ltr',
    format: '',
    indent: 0,
    version: 1,
  },
}
```
Create a helper function `lexicalParagraph(text: string)` at the top of the seed file that generates this structure for a single paragraph.
