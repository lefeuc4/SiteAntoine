## Spacing Scale

Declared values (must be multiples of 4). All values map to Tailwind v4 utilities (p-1 = 4px, p-2 = 8px, etc.).

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge padding, inline spacing between icon and label |
| sm | 8px | Compact padding inside tags, tight row gaps |
| md | 16px | Default padding inside cards, button horizontal padding |
| lg | 24px | Section internal padding, card gaps in grid |
| xl | 32px | Horizontal page padding on mobile, gap between major sections on mobile |
| 2xl | 48px | Section vertical padding on tablet |
| 3xl | 64px | Section vertical padding on desktop, hero vertical padding |

Exceptions:
- Touch targets (hamburger, nav items): minimum 44px height — use `min-h-[44px]`
- Hero split layout image column: full viewport height on desktop (`h-screen` or `min-h-screen`)
- Before/after slider handle: 40px width touch zone

## Component Contracts

### Header

- Sticky (`position: sticky; top: 0`), `z-index: 50`
- Transparent background when at top of page (over hero image), transitions to `bg-blanc-pur` with `shadow-sm` on scroll
- Transition: `--transition-slow` (300ms ease-out) on background and shadow
- Logo left, nav center (desktop), CTA button right
- CTA button: `bg-bleu-electrique text-blanc-pur` — always visible (D-04)
- Mobile: logo left, hamburger right (44px touch target)
- Hamburger opens full-screen overlay (`bg-bleu-nuit/95`) with nav links centered vertically
- Nav links: 5 items max — Accueil, Mon Histoire, Mes Services, Les Programmes, Resultats

### Footer

- Minimal (D-03): `bg-bleu-nuit text-blanc-pur`
- Three zones: logo/tagline left, nav links center, contact link right
- Bottom bar: mentions legales link, copyright
- Padding: `2xl` (48px) vertical on desktop, `xl` (32px) on mobile

### Page: Accueil (D-05, D-06)

**Hero — split layout:**
- Desktop: 50/50 split. Left: display headline + subheading + primary CTA button. Right: placeholder image block (full height, `bg-gris-ardoise` with coaching icon overlay)
- Mobile: stacked. Image first (200px height, full width), text below
- Headline: `text-4xl font-heading font-black text-bleu-nuit`
- Subheading: `text-base font-body text-gris-ardoise`
- CTA button: `bg-bleu-electrique text-blanc-pur` rounded, `px-6 py-3`
- Hero vertical padding: `3xl` (64px) top/bottom

**Sections order (D-06):**
1. Services apercu — 3-4 cards in grid (icon + titre + description courte), no CTA on cards
2. Presentation — paragraph + lien "Mon Histoire" (text link, bleu-electrique)
3. Resultats en vedette — 2-3 result cards (same card pattern as page Resultats)
4. Bandeau CTA final — full-width dark section (`bg-bleu-nuit`), headline + CTA button centered

### Page: Mon Histoire (D-07)

**Timeline verticale:**
- Single centered vertical line (`border-l-2 border-bleu-electrique`)
- Alternating left/right on desktop, all right on mobile
- Each step: dot marker (8px circle, `bg-bleu-electrique`) + year label (`text-sm font-body text-gris-ardoise`) + titre (`text-xl font-heading`) + texte body
- Spacing between steps: `3xl` (64px)

### Page: Mes Services (D-08)

**Grid de cartes:**
- Desktop: 3 columns. Tablet: 2 columns. Mobile: 1 column
- Each card: `bg-blanc-pur` + `border border-gris-ardoise/20` + `rounded-xl` + `p-6`
- Card anatomy: icon (lucide, 32px, `text-bleu-electrique`) + titre (`text-xl font-heading`) + description (`text-base font-body`) + CTA text-link ("En savoir plus →", `text-bleu-electrique`)
- Hover: `shadow-md` + translate-y -2px, `--transition-base` (200ms)

### Page: Les Programmes (Claude's Discretion — D-09)

**Grid de cartes adapte aux champs Payload:**
- Same grid layout as Mes Services (3/2/1 col)
- Card anatomy: image de couverture (placeholder color block, aspect-ratio 16/9, `rounded-t-xl`) + corps (`p-6`): titre + badge duree (`text-sm text-vert-energie bg-vert-energie/10 rounded-full px-3 py-1`) + description (`text-base`) + liste objectifs (bullet list, `text-sm`) + tag public cible (`text-sm text-gris-ardoise`)
- No CTA on programme cards in Phase 2 — no detail page exists yet

### Page: Resultats (D-10, D-11, D-12, D-13)

**Grid de cartes avec slider comparatif:**
- Desktop: 2 columns. Tablet: 2 columns. Mobile: 1 column
- Each card: `bg-blanc-pur rounded-2xl overflow-hidden shadow-sm border border-gris-ardoise/10`
- Card anatomy:
  1. Before/after slider (top, aspect-ratio 4/3): two images stacked with CSS `clip-path` driven by range input overlay. Slider handle: 40px wide, `bg-bleu-electrique`, vertical drag line, touch/mouse events
  2. Citation: `italic text-base font-body text-bleu-nuit` with `"` decorators, `p-4`
  3. Meta row: `text-sm font-body text-gris-ardoise` — prenom + programme suivi + duree

**Slider interaction:**
- HTML `<input type="range">` overlaid on image pair, transparent, full-width, draggable
- Left image (avant) revealed via `clip-path: inset(0 X% 0 0)` where X = 100 - slider value
- Right image (apres) full width behind
- Handle position driven by CSS custom property updated on input event
- No JS library — native range input + CSS custom property

## Animations & Motion

Source: Phase 1 D-05, `--transition-fast/base/slow` defined in `globals.css`

| Interaction | Animation | Token | Notes |
|-------------|-----------|-------|-------|
| Hover on card | Shadow increase + translate-y -2px | `--transition-base` 200ms | All clickable cards |
| Hover on button | Brightness increase (`hover:brightness-110`) | `--transition-fast` 150ms | CTA buttons |
| Header background on scroll | bg transparent → bg-blanc-pur + shadow | `--transition-slow` 300ms | JS scroll listener |
| Scroll reveal | Fade in + translate-y 16px → 0 | `--transition-slow` 300ms | Section entries, use `IntersectionObserver`, no library |
| Before/after slider drag | Instant clip-path update | none (real-time) | Driven by range input event |
| Mobile nav open/close | Opacity 0→1 + scale 0.97→1 | `--transition-base` 200ms | Full-screen overlay |

Scroll reveal: apply class `.reveal` on elements, toggle `.revealed` via IntersectionObserver. Initial state: `opacity-0 translate-y-4`. Final state: `opacity-100 translate-y-0`. Threshold: 0.15. No external library.

## Data Integration Contract

Pages consume Payload local API (no REST/GraphQL overhead — direct `getPayload()` call in Server Components).

| Page | Payload collection | Fields consumed |
|------|--------------------|-----------------|
| Accueil — resultats en vedette | Resultats | prenom, citation, photosAvant[0], photosApres[0], programmeS suivi, duree (first 2-3 docs) |
| Accueil — services apercu | PageContent (section: services-apercu) | titre, items[].titre, items[].description, items[].icone |
| Les Programmes | Programmes | titre, description, duree, objectifs, publicCible, imageCouverture, ordre |
| Resultats | Resultats | tous les champs (prenom, citation, histoire, programmeS suivi, duree, photosAvant[], photosApres[]) |
| Mon Histoire | PageContent (section: mon-histoire) | items[].annee, items[].titre, items[].texte |
| Mes Services | PageContent (section: mes-services) | items[].titre, items[].description, items[].icone |

Images: served via `next/image` with Vercel Blob URLs. Placeholder: `bg-gris-ardoise` color block with `animate-pulse` skeleton while loading.
